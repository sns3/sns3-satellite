Detailed Description

Produce capacity request statistics of Random Access Slotted ALOHA from a satellite module simulation.

Statistics logs every capacity request made in a simulation. Statistics consists of tuples: time, node id, rc id, type (RBDC/VBDC), kbps (RBDC) / bytes (VBDC), queued bytes.
